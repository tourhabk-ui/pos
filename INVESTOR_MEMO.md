## 1. Executive Summary

TourHab — операционная система для туриста в дикой местности. Первый продукт, заточенный под Камчатку, с архитектурой для масштабирования на любой регион.

**Проблема:** Туристы в дикой местности (Камчатка, Алтай, Байкал, Саяны) сталкиваются с тремя рисками:
1. **Потеря связи** — нет интернета, не знают где находятся
2. **Отсутствие информации** — не знают текущий статус вулканов, погоды, троп
3. **Безопасность** — нет SOS с координатами, никто не знает куда они пошли

**Решение:** PWA-приложение с офлайн-картой, AI-ассистентом (Kuzmich), SOS-системой, регистрацией маршрута и живыми данными. Работает без интернета.

**Ров:** Локальная экспертиза + офлайн-технологии. Walvee и AllTrails не работают в России, не имеют данных по Камчатке, не делают офлайн-контент на русском.

**Статус:** Продукт в продакшене (tourhab.ru). 1423 маршрута в БД, 464 API endpoint, 132 миграции, 195K+ строк кода.

## 3. Дорожная карта

### Фаза 1 — Технический фундамент (апрель-май 2026) ✅ почти закрыта

- Офлайн-карта с GPS ✅
- Kuzmich с геоконтекстом ✅
- SOS с координатами ✅
- Регистрация маршрута ✅
- Live-данные: КВЕРТ вулканы, Яндекс.Погода ⏳

**Результат:** Турист может открыть приложение в Налычево без связи и получить полезное действие.

### Фаза 2 — Контент топ-100 (май-июнь 2026)

- 300 глубоких маршрутов (фото, описание 200-300 слов, сезон, опасности)
- 1,100 базовых маршрутов (категория + 1-2 предложения)
- Manus baseline-30 (UNESCO + заповедники + must-see)
- search_count → data-driven top-100

**Результат:** Карта живая. 300 «глубоких» + 1,100 «базовых».

### Фаза 3 — Запуск сезона (май-июнь 2026)

- Релиз публичный
- 100-300 установок PWA
- 5-10 активных операторов
- Посевы в туристические чаты
- Договорённости с операторами

### Фаза 4 — UGC + lock-in (лето-осень 2026)

- Фото с GPS-привязкой
- Заметки на маршруте
- GPX upload
- Отчёты после поездки (автогенерация)
- Push-уведомления (FCM)
- Чек-ин на вершинах

**Прогноз:** 200 установок × 30% активных = 60 туристов → 6-12 UGC за сезон

### Фаза 5 — Возврат привычки (зима 2026/27 → лето 2027)

- UGC → SEO-трафик («отзыв о туре на Авачинский 2026»)
- 800 глубоких маршрутов (из 1,300 базовых)
- Live-данные стабилизированы
- RAG обогащён реальными разговорами
- 1,000-2,000 установок

**Результат:** Must-have status. Туристы возвращаются с приложением, приводят друзей.
